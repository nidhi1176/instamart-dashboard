

# 📊 Swiggy Instamart Sales Dashboard (Power BI)

## 🔍 Overview

This project presents an **interactive Power BI dashboard** built to analyze sales performance, customer behavior, and operational insights for a retail/grocery platform (inspired by Swiggy Instamart).

The dashboard transforms raw data into **actionable insights** using data modeling, DAX, and visualization techniques.

---

## 🎯 Key Objectives

* Analyze **total sales, orders, and average performance**
* Understand **sales distribution by city tier and shop type**
* Track **year-wise order trends**
* Identify **top-performing product categories**
* Evaluate **shop size and rating impact on sales**

---

## 📌 Key Metrics

* 💰 Total Sales: **1.20M**
* 🛒 Total Orders: **8523**
* 📊 Average Sales: **141.01**
* ⭐ Average Rating: **3.92**

---

## 📈 Dashboard Features

### 🔹 Sales Analysis

* Total Sales by **Shop City Tier (Tier 1, Tier 2, Tier 3)**
* Total Sales by **Shop Size (Small, Medium, High)**

### 🔹 Order Insights

* Total Orders distribution across **City Tiers**
* Year-wise trend of orders (2021–2024)

### 🔹 Category Performance

* Top categories like:

  * Fruits & Vegetables 🍎
  * Snack Foods 🍪
  * Household 🏠
* Helps identify **high-demand product segments**

### 🔹 Shop Type Analysis

* Comparison of:

  * Grocery Stores
  * Supermarket Type 1, 2, 3
* Includes **Total Orders & Sales contribution**

### 🔹 Interactive Filters

* Year filter (2021–2024)
* City Tier filter
* Shop Size filter
* Dynamic slicers for better exploration

---

## 🛠️ Tools & Technologies

* **Power BI** (Data Visualization)
* **DAX** (Data Analysis Expressions)
* **Power Query** (Data Cleaning & Transformation)
* **Data Modeling** (Relationships, Star Schema)

---

## 🧠 Key Insights

* Tier 3 cities contribute the **highest sales (0.47M)**
* Supermarket Type 1 generates the **maximum orders & revenue**
* Fruits & Vegetables and Snack Foods are **top-performing categories**
* Medium-sized shops contribute the **largest share of total sales**
